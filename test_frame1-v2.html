Key Implementation Points for Frame #1 v2:

  • Starting Point: Frame begins at (0,0) with no canvas padding
  • Black Border: 14px thick black rounded rectangle (14px radius)
  • White QR Area: 272×272px with 4px border radius, positioned 14px from edges
  • QR Code: Approximately 240×240px, centered within white area
  • Label: "SCAN ME" text below QR code with 8px gap
  • Total Height: 332px (272px QR area + 14px top padding + 14px bottom padding + 8px gap + label height)
  • No External Padding: Unlike "no frame" option which has 30px padding, Frame #1 fills the canvas from edge to edge

Note: The actual QR code size inside the white area would be calculated dynamically based on the QR code complexity and error correction level, but is typically around 80-90% of the white area size.
