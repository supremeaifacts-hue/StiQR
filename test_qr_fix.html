QR Code Connection Test

Problem Analysis

Issue: Phone can't connect to localhost:3000 when scanning QR code

Root Cause 1: QR code contains localhost:3000 which only works on your computer

Root Cause 2: /track/:id route was mounted at /api/track/:id instead of root level

Solution 1: Use computer's IP address (192.168.1.104:3000) in QR code

Solution 2: Added root-level /track/:id route in backend
