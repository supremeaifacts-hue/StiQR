Diagnostic Information

Canvas Drawing Method

Frames are drawn using:

ctx.strokeStyle = color;
ctx.lineWidth = 1;
// Add 0.5 to coordinates to avoid anti-aliasing
ctx.strokeRect(x + 0.5, y + 0.5, width, height);

The + 0.5 offset ensures crisp 1px lines by aligning to pixel centers.

Common Issues

  • Missing right/bottom edges: Canvas draws from (x,y) to (x+width, y+height), so the right edge is at x+width and bottom at y+height
  • Blurry lines: Drawing at integer coordinates can cause anti-aliasing
  • Clipped edges: If width/height includes the border thickness, edges may be clipped
